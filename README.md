Weather Dashboard

A simple React-based Weather Dashboard that allows users to search for any city and view its current weather information. This project uses the OpenWeatherMap API to fetch live weather data.
Features

Search for any city worldwide.

View current weather including:

Temperature (°C)

Weather description

Humidity

Wind speed

Atmospheric pressure

Loading indicator while fetching data.

Error handling for invalid city names.
Technologies Used

React – Frontend library

OpenWeatherMap API – For real-time weather data

CSS – Styling with gradient background and card layout

Vite – Development server and build tool
